# 你的 Loading

> 开箱即可用的 loading，
> 说明：vue3-loading 是一个方便在 Vue 3 项目中使用的加载指示器组件的 npm 插件。它允许您轻松地在项目中添加加载动画，提升用户体验。

# 🌍 安装

```javascript
npm install vue3-loading-plug
```

## 🛹 使用方法

内置七款 Loading ，可以配置 loadingType 属性进行指定。没有 loadingType 属性时，默认每次刷新页面随机展示。
